# SHA in C++

## Syntax
On Linux/Max run the program using `./sha`.\
On Windows run it using `sha.exe`.

For some help: 
```c++
./sha -h
      --help


Ordinary syntax:
```c++
./sha -1    -a        -s        "string or filename"
      -224  --ascii   --string
      -256  -e        -f
      -384  --hex     --file
      -512  -b
            --binary
```
